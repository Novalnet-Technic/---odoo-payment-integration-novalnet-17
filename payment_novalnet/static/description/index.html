Inline Credit Card Payment

Capture credit/debit card payments directly from the checkout page. There is no need to redirect the user to the portal to make a payment.

Novalnet processes all payments via dedicated escrow accounts guaranteeing assured payouts and eliminating risks arising from bankruptcy, etc. In addition Novalnet offers Guaranteed Payments to protect online businesses around the world from disputes and frauds by taking over the client's liability..

How to configure

Step 1 Download the module and place the module in apps folder.

Step 2 Install this module in your Odoo instance.

Step 3 Open Website > Configuration > Payment Providers and open Novalnet providers

Step 4 Fill the API keys and credentials. (You will find these keys from your Novalnet Admin portal dashboard.)

Step 5 Navigate Invoicing > Configuration > Payment Providers and configure the required payment settings in the Novalnet Admin Portal.

DONE That's it you are ready to accept payments.
